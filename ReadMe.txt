***** Steps to run the project: ****** 

1- double click on Tiba_Server.exe to run the server on port 2525

2- now you can join from clients to the server since it's no running.

3- double click on Tiba_Client 

4- insert your name and just hit connect

5- start write and send message 

6- server should log every message and send it back to all clients

7- if you want to join another client

8- just repeat from step 3 to step 6
